Solution - Scenario Author

Team Kiwi collaborated with Western Governors University (WGU) to explore methods to offer engaging, authentic and relevant formative activities to WGU students.

The Scenario Author is an authoring tool that makes it easy to create branching scenario activities with videos, text and embedded quizzes.

The authoring tool supports best practices of instructional design, helping designers to make sure the activities include ample feedback and target relevant learning objectives.

Instructional developers can create a branching scenario activity by dragging and connecting video, text and question blocks onto the canvas.

For example, a teacher is required to do a formative activity in the format of video embedded quiz. Once logged in, she would be able to associate the target skill with the activity. Then she could drag the video blocks and import videos to initiate the activity. Afterwards, she used the multiple choice quiz block to set up all the options for students. With these options available, she would build according to video blocks for each branching path. That’s it. Let’s start exploring more!
